## Introduction

This software is a localized version based on [SteamEdit](https://steamedit.tg-software.com/). Special thanks to [batvbs](https://gitee.com/batvbs/steam-editor) for paving the way and providing a valuable reference for the translation.

Due to Steam's update to the `appinfo.vdf` format, older versions of SteamEdit have become incompatible. Since the previous translator, **@batvbs**, has been inactive for two years, I decided to take matters into my own hands to ensure the tool remains functional.
